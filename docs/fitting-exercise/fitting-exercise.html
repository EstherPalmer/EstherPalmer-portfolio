remove all OCC=2 entries, keep only OCC=1

df2 <- subset(df, (OCC == 1))

Exclude all Time=0, found sum of DV for each ID, create a new data frame with only time 0 and a new variable Y that contains amount of drug delivered over the whole experiment

df3 <- subset(df2, (TIME != 0))
df4 <- df3 %>% group_by(ID) %>% summarise(Y = sum(DV))
df5 <- subset(df2, (TIME == 0))
df6 <- left_join(x=df5, y=df4, by=c("ID"))

df7 <- select(df6,Y, DOSE, AGE, SEX, RACE, WT, HT)

The black points represent the original model predictions, and most of the points lie close to the 45° line, this indicates that there is not much variability between observed and predicted values. The blue points represent the median predictions from the bootstrap samples, they overlap with the original predictions. This overlap shows that the model is consistent across resampled datasets. The red vertical lines indicate the 89% confidence intervals, which are small for most individuals demonstrating low uncertainty, but are bigger for a few individuals. The pattern shows that the model represents most of trends in the data, and the bootstrap confirms the consistency and uncertainty of the predictions.
