Then to load data

data_location <- here::here("fitting-exercise", "Mavoglurant_A2121_nmpk.csv")
df <- read.csv(data_location)

Plotting outcome (DV) as a function of time colored by dose and using ID as a grouping factor

p1 <- ggplot(df, aes(x=TIME, y=DV, group=ID, color=DOSE)) + geom_line()
plot(p1)

remove all OCC=2 entries, keep only OCC=1

df2 <- subset(df, (OCC == 1))

Exclude all Time=0, found sum of DV for each ID, create a new data frame with only time 0 and a new variable Y that contains amount of drug delivered over the whole experiment

df3 <- subset(df2, (TIME != 0))
df4 <- df3 %>% group_by(ID) %>% summarise(Y = sum(DV))
df5 <- subset(df2, (TIME == 0))
df6 <- left_join(x=df5, y=df4, by=c("ID"))

Keep only variables Y, Dose, Age, Sex, Race, WT, HT. Convert Race and Sex to factors

df7 <- select(df6,Y, DOSE, AGE, SEX, RACE, WT, HT)
df7$SEX <- as.factor(df7$SEX)
df7$RACE <- as.factor(df7$RACE)

The roc_auc is 1 for the last model which is super interesting. I guess that means that all the data put together are very good at predicting sex. This makes sense given that a lot of the things in the model are correlated with sex (height/weight, and doeses tend to be different for men/women). In model 3, dose alone had an ROC of .6, so it’s probably a decent predictor of sex.
